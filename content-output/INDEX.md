## How to Publish

1. **Collection Pages (Phase 1)**: Go to Shopify Admin > Products > Collections > [collection] > Edit description. Paste the HTML content. Add the FAQ schema JSON-LD to the collection template in your theme.
2. **New Blog Posts (Phase 2 & 4)**: Go to Shopify Admin > Online Store > Blog Posts > Add blog post. Paste the HTML into the content editor (switch to HTML mode). Set the meta title and meta description from the comments at the bottom of each file.
3. **Updated Blog Posts (Phase 3)**: Navigate to the existing blog post URL listed in the file comments. Replace the existing content with the new HTML.
